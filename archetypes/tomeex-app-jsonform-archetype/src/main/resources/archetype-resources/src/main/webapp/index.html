🚀 Scenario API Endpoints

GET /api/scenario/config/{moduleName}

Get full processed configuration for a module

Parameters: role (optional), maintenance (optional)

GET /api/scenario/list/{moduleName}

List all scenarios in a module

GET /api/scenario/{moduleName}/{scenarioKey}

Get specific scenario details

📁 Directory Structure

Place your scenario modules in:

${symbol_dollar}{catalina.base}/scenarios/

Each module should follow the structure:

ModuleName/
  ├── Config/
  │   └── json/
  │       ├── ModuleName.json
  │       └── forms/
  │           ├── ENTITY_FORM.json
  │           └── ENTITY_LIST.json
